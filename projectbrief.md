# Project Brief

## Purpose

Comicr is a web application for discovering, reading, and managing digital comics.

## Scope

- User authentication and profiles
- Comic browsing and search
- Reading interface with bookmarking
- User library and favorites
- Admin panel for comic uploads

## Requirements

- WHEN a user signs up, THE SYSTEM SHALL create a new profile and send a welcome email.
- WHEN a user searches, THE SYSTEM SHALL return comics matching the query.
- WHEN a user reads a comic, THE SYSTEM SHALL track reading progress.
- IF a user is not authenticated, THEN THE SYSTEM SHALL restrict access to the library and reading features.

## Stakeholders

- End users (comic readers)
- Admins (comic uploaders)
- Product owner
- Development team
