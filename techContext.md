# Tech Context

## Stack

- Next.js 14 (app-router, server components)
- TypeScript
- Playwright (E2E)
- Jest (unit)
- PostgreSQL, Redis

## Setup

1. Clone repo
2. `npm install`
3. Copy `.env.example` to `.env` and fill in secrets
4. `npm run dev`

## Dependencies

- next
- react
- @playwright/test
- jest
- pg
- redis
